Godot Piano Transcription Helper
version 1.0

This is a tool that analyses music files and displays the magnitude of frequencies for each piano note.

Shortcut keys:
O: open file
Space: play/pause
Arrow keys: jump 1 second/5 seconds
R: jump to start of track
L: loop on/off
M: mute/unmute
S: smoothing on/off
F: filtering on/off
H: hide UI/hide keyboard/unhide
F11: full screen/windowed
C: switch colour modes
